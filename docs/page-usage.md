# ページの使い方

このアプリケーションは静的ページ + API で動作します。ここでは「画面として見るページ」の使い方をまとめます。

## トップページ `/`

- 役割: 対戦履歴の一覧表示とゲーム開始への入口
- 使い方:
  - 画面右（リンク）にある「対戦する」を押すと対戦画面へ移動します。
  - 下部の表に、過去の対戦結果が一覧で表示されます。
- 表の内容:
  - `黒を打った回数`: 黒が置いた総数
  - `白を打った回数`: 白が置いた総数
  - `勝った石`: 勝者（黒/白/引き分け）
  - `対戦開始時刻`: ゲーム開始時刻
  - `対戦終了時刻`: ゲーム終了時刻

## 対戦ページ `/game/`

- 役割: リバーシの対戦を行う画面
- 使い方:
  1. ページを開くと自動で新しいゲームが開始されます。
  2. 盤面（8x8）上の空きマスをクリックすると、石を置ける場所であれば次の手に進みます。
  3. 置けない場合は変化しません（有効手のみ進行）。
- 画面の表示:
  - 「次は黒の番です」など次の手番が表示されます。
  - 打てる手がない場合は「◯の番はスキップです」と表示されます。
  - ゲーム終了時は「引き分けです」または「◯の勝ちです」と表示されます。

## 備考

- ページは静的ファイルとして配信され、データの取得や手の登録は API 経由で行われます。
- ゲームの進行は「対戦ページ」内だけで完結します。
